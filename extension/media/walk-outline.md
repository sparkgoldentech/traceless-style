# Outline · Workspace search · Status bar

Three structural views over your styles:

- **Outline** (`Ctrl+Shift+O`): shows every `tl.create` group key in the current file, with `_dark` / `_hover` blocks nested.
- **Workspace symbols** (`Ctrl+T`): searches every group across the whole project. Type `card` → see every file that has a `card:` group.
- **Status bar**: bottom-left of the editor shows `tl 12 groups · 47 rules` for the active file. Click it to jump to the outline.
- **Inlay hints**: ghost-text rule counts next to every group key (`btn: ⟨3 rules⟩`).

For very large style files, this turns navigation from "scroll and grep" into "fuzzy-search and click."
